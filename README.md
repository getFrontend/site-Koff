# site-Koff
Website of online furniture store

[Live demo](https://getfrontend.github.io/site-Koff/dist/)

## Built with:
* Mobile-first workflow
* HTML Semantic Tags
* BEM (Block, Element, Modifier)
* SCSS
* JavaScript
* CSS Flexbox
* CSS Grid
* Fluid Typography
* Fluid Space


### Js modules
* Navigo (routing, hooks)
* Axios
* Swiper
